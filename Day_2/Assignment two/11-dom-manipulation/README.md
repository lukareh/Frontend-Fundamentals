Concepts
- DOM selection (getElementById, querySelector)
- Event listeners (addEventListener)
- Element manipulation (textContent, innerHTML)
- Creating elements (createElement, appendChild)
- Removing elements (remove, removeChild)

Key DOM Methods Used

- getElementById() - Select element by ID
- querySelector() - Select element using CSS selector
- addEventListener() - Attach event handlers
- createElement() - Create new HTML elements
- appendChild() - Add child elements
- remove() - Remove element from DOM
- textContent - Get/set text content
- innerHTML - Get/set HTML content
