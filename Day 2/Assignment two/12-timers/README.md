Timers

Concepts
- setInterval() - Execute function repeatedly at specified intervals
- clearInterval() - Stop interval execution
- setTimeout() - Execute function after delay
- Timer management

Assignments

Countdown Timer
Create a countdown timer using setInterval.

Open: countdown-timer.html

Features

The countdown timer includes:
- Customizable minutes and seconds
- Start/Pause/Reset controls
- Beautiful gradient UI
- Pulse animation for last 10 seconds
- Alert when timer completes
- Real-time status updates

How to Use

1. Open countdown-timer.html in your browser
2. Set the desired minutes and seconds
3. Click "Start" to begin countdown
4. Use "Pause" to pause the timer
5. Use "Reset" to reset to initial time

Timer Methods

- setInterval(callback, delay) - Executes callback every 'delay' milliseconds
- clearInterval(intervalId) - Stops the interval
- setTimeout(callback, delay) - Executes callback once after 'delay' milliseconds
- clearTimeout(timeoutId) - Cancels the timeout
